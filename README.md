# 🎬 Netflix UI Clone

A responsive **Netflix Landing Page Clone** built using **HTML and CSS**.
This project recreates the look and feel of Netflix's homepage UI for learning and practicing **frontend development**.

---

## 🚀 Features

* Netflix-style **Hero Section**
* **Movie Cards Slider**
* Clean and simple **UI layout**
* **Hover effects**
* Organized **assets folder structure**
* Basic **responsive design**

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
